## 📝 Bug Nasıl Raporlanır?

1. Bu dosyayı kendi branch'inizde düzenleyin
2. "Aktif Buglar" tablosuna yeni satır ekleyin
3. Mümkünse screenshot veya log ekleyin (docs/bugs/ klasörüne)
4. PR açın → `develop` branch'ine
5. Fix yaptıysanız, bug'ı "Çözülen Buglar"a taşıyın

### Commit Formatı
```
fix(bug-no): kısa açıklama
```
